# AI Job Impact 2030 - Executive Dashboard & Analytics

An advanced, interactive Power BI analytics suite designed to evaluate the projected impacts of Artificial Intelligence on global job markets, compensation structures, future demand indices, and remote work viability by the year 2030.

## 📊 Project Overview
This project transforms raw data into an executive-level strategic asset, enabling leadership teams to analyze macro workforce shifts. The report transitions seamlessly from a high-level corporate landing experience to granular cross-filtered metric modules.

### Core Analytical Pages:
1. **Main Page / Navigation Hub**: A clean corporate gateway featuring stylized, image-mapped navigation assets to move seamlessly through sub-modules.
2. **Executive Dashboard**: A comprehensive overview displaying critical macro-KPIs, multi-dimensional trends (AI Risk vs. Salary by Industry), and sector-by-sector breakdown analytics.
3. **Global Risk Heatmap**: A spatial analytics module charting spatial density and automation impact variances across international boundaries (US, UK, Germany, UAE, India, Singapore, etc.).
4. **Advanced KPI Quadrant**: A highly customized, granular performance matrix evaluating dynamic variances and baseline ratios for Compensation, Role Risk, Future Demand Scores, and Remote Work Possibility.
5. **Data Matrix Module**: A deep-dive structured pivot view detailing micro-level industry attributes, employee concentrations, and specific job-title vulnerabilities.

---

## 🛠️ Tech Stack & Advanced Features
* **Business Intelligence Engine:** Power BI Desktop
* **Data Modeling & Architecture:** Star schema optimizations, row-level context overrides, and dynamic temporal baseline calculations.
* **Advanced DAX & Custom KPIs:**
  * Context-aware percentage calculations factoring asynchronous data structures (e.g., parsing categorical `"Yes"`/`"Hybrid"` strings to measure real-time remote infrastructure viability ratios).
  * `CALCULATE` and `ALL` baseline evaluations to compute dynamic variance ratios against rigid global benchmarks (e.g., comparing local filtered averages directly against stable global medians within a unified card element).
* **Interactivity Framework:** Full Slicer Synchronization maps across separate canvas layers, alongside user-centric custom-state navigation mechanisms.

---

## 🔑 Key Strategic Insights Captured
* **The High-Value Compensation Paradox:** High-salaried specialized roles generally display stronger insulation against total automation, trending toward AI-augmentation and productivity multipliers rather than baseline role reduction.
* **Remote Flexibility Disparity:** Data exposes a clean divergence between remote-capable knowledge sectors and non-remote physical execution operational fields.
* **Demand Shifts:** Identifies critical future growth sectors where technical demand indexes outpace legacy corporate positions.
